# FIFA Data Reuse Permission Request - Draft

Do not send without the repository owner's approval.

## Subject

Permission request for limited FIFA World Cup 2026 Post Match Summary Report data
in a free public web challenge entry

## Draft

Hello FIFA Training Centre team,

I am building a free, publicly accessible educational web prototype for the DAKER
World Cup tactics web challenge in Korea. The prototype lets a user create one
pressing instruction and compare it with clearly labeled, observed match
aggregates. It will not predict match outcomes or claim FIFA endorsement.

I would like written permission to use a small subset of metrics from the
following FIFA World Cup 2026 Post Match Summary Reports:

- [list exact match IDs and PDF URLs after product selection]
- proposed fields: formations; phases of play percentages; line height, team
  length, and width; through/around/over line-break attempted/completed totals;
  pressure direction totals; forced turnovers; and ball recovery time.

The intended use is:

- a free deployed website with no login, payment, or advertising;
- a public GitHub repository containing the extraction/transform code and a small
  derived JSON file;
- a public YouTube demonstration;
- submission to a prize-bearing web challenge;
- original graphics only. FIFA logos, flags, report pages, screenshots, and chart
  artwork will not be copied.

Could you please confirm whether FIFA grants permission for:

1. manual or automated extraction of the named fields;
2. display of those metrics in original interactive graphics;
3. transformation into the limited derived JSON needed by the browser app;
4. publication of that derived JSON and transformation code on GitHub;
5. display in the YouTube demo and during competition judging;
6. the prize-bearing nature of the submission;
7. the required attribution wording, link, term, and geographic scope?

Proposed attribution:

> Source: FIFA Training Centre, FIFA World Cup 2026 Post Match Summary Report,
> [match and date], [pages], accessed [date]. Unofficial educational analysis;
> not affiliated with or endorsed by FIFA.

If any requested use is not permitted, please identify which of display,
transformation, automated extraction, public redistribution, or competition use
must be removed.

Thank you.
