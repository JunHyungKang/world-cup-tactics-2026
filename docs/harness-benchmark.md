# Agent Harness Benchmark

## Retained patterns

| Source | Retained pattern | Adaptation |
|---|---|---|
| SitRep ActionGate | Observable user loop, deployment preflight, adversarial scenario | Manager choice-to-feedback loop, clean-browser judging path |
| NeuroGolf | Layered validator, immutable submission evidence, deadline guardrail | Build/browser/URL checks, planning PDF hash, final commit freeze |
| NNRC | Expected-value experiment gate, data provenance, exact handoff | Product concept gate, licensed football-data manifest, P1 resume state |
| Media Statistics | Claim honesty, source audit, single-thesis lane | Prototype/data-backed labeling and one differentiating manager fantasy |

## Not ported

Kaggle slot watchers, ONNX/model workflows, semantic knowledge graphs, autonomous
supervisors, statistical survey machinery, and SitRep-specific signing/deployment
contracts. They add maintenance without strengthening this submission.

## Invariant

Every primary demo beat must connect a direct user manipulation to a visible,
tested consequence. Any claimed football insight additionally needs accepted
data provenance, a reproducible calculation, and a clearly stated limitation.
