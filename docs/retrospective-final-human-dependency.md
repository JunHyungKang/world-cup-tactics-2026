# Retrospective — Final Human Dependency Audit

Date: `2026-07-19`

## Evidence

The official final contract requires a public keyless app, public GitHub source,
a public YouTube demo, major-browser operation, and a demo of the core manager
loop. It does not require recruited participants, physical VoiceOver approval, a
named human visual reviewer, or a named human demo-loop reviewer.

The prior harness nevertheless required `voiceover MANUAL-PASS`, `visual
MANUAL-PASS`, a human `demo-loop PASS`, and real-human reviewer identities. That
made unavailable evidence a submission blocker and encouraged an unsafe choice:
fabricate a human-like approval or miss the deadline.

## Smallest durable correction

Keep all machine, public-surface, provenance, deadline, history, and receipt
checks. Replace only the unsupported manual-review predicates with one canonical
SHA-bound independent-agent final-artifact review. The review must identify a
distinct `/root/<task>` non-creator, bind the exact release, browser report/test
source, video/demo manifest, completion times, exact criteria, and zero findings.
Preflight must independently recompute four-project screenshots, canonical video
frame pixels, audio/caption audits, exact row shape, and evidence chronology; a
self-declared empty PASS review is explicitly insufficient. Both the review and
owner upload attestation remain auditable in content-addressed tracked files.
The reviewer task name is procedural, not cryptographic spawn provenance; this
limitation stays explicit until the runtime exposes a signed task receipt.

The ledger truthfully records `voiceover UNAVAILABLE-NO-CLAIM`. Agent inspection
is artifact QA only; it cannot create human accessibility, usability,
preference, or memorability evidence.

- confirm the logged-in DAKER account and submission surface;
- upload and final-save the exact planning PDF;
- authorize credentialed deployment and public repository/video publication;
- attest that the exact local candidate was uploaded to YouTube, which transcodes

No participant recruitment or additional human reviewer is on the critical path.
